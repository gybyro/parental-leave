# Bubblify (Next.js Edition)
**By: Výrin Dreki**


## Project Overview
Bubblify is a modern e-commerce platform for high-quality bubble products. 
This project was built using Next.js 15 (App Router), TypeScript, and Tailwind CSS.


### Custom Changes
- **Server Data**: I have expanded the `products.json` and server entries with additional items to better showcase the catalog, please run the server attatched to this project instead.
- **UI/UX**: I adjusted the grid layout to better accommodate the new product variety because I think a 3 column grid is too much space for each bubble (please dont dock me points thanks in advance).


## How to run the project
The backend is included in this submission. Open a terminal in the server folder and run:

```bash
npm install
npm run build
npm start
```


run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
